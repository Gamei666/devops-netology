1. cd - встроенная команда оболочки, это подтверждается с помощью type cd (cd is a shell builtin), а также тем, что при вводе command -v cd нам не выдаст какого-то отдельного пути к команде, а просто будет cd.
2. Альтернативой grep <some_string> <some_file> | wc -l может являться grep -c <somestring> <somefile>, то есть в самом grep уже есть возможность выводить количество совпадающих строк. 
3. родителем всех команд является systemd, который является подсистемой инициализации команд и служб.
4. Команда для перенаправления вывода ошибки команды ls на другую сессию будет выглядеть так ls <несуществующая директория> 2>/dev/pts/х (где х - номер терминала)
