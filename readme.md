Длина журнала History задаётся переменной history-size, это описывается в строке 3281

ignoreboth задаёт не сохранять в журнале истории строки, которые начинаются с пробела (ignorespace), а также строки, которые уже были в журнале (ignoredups).

Фигурные скобки применимы в тех случаях, когда нам необходимо перечислить несколько параметров для команды.

Для создания с помощью touch 100000 файлов можно использовать фигурные скобки, например touch filename{1..100000}, 300000 файлов таким образом создать нельзя, посколько таким образом будет нарушен максимальный размер одного аргумента.

Двойные квадратные скобки [[]] проверяют истинность или ложность выражения и выдают значение 0 если истина и 1 если ложно. [[ -d /tmp ]] проверит существование директории tmp и вернёт 0, так как такая директория существует (проверено с помощью if [[ -d /tmp ]]  ; then echo 0; else echo 1 ;fi ). 

export PATH=/tmp/new_bash/:$PATH позволяет добиться следующего вывода type -a bash:
bash is /tmp/new_bash/bash
bash is /usr/bin/bash
bash is /bin/bash

Планирование команд с помощью at и batch отличается тем, что в at устанавливается конкретное условие выполнения команды, а в batch выполнение начинается в зависимости от загруженности системы.
